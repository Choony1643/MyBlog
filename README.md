# My Blog

Personal blog built with [Jekyll](https://jekyllrb.com/) and [Reverie](https://github.com/amitmerchant1990/reverie) theme, hosted on GitHub Pages.

> [Visit my blog](https://Choony1643.github.io/MyBlog/)

---

## About

This is my personal space for documenting what I learn and experience — from SAP operations and AI exploration to calligraphy and electric guitar practice.

---

## Tech Stack

- **Static Site Generator**: Jekyll
- **Theme**: Reverie
- **Hosting**: GitHub Pages
- **CI/CD**: GitHub Actions

---

## Features

- Clean and minimal design
- Fully responsive
- Markdown blogging
- Syntax highlighting
- Search across posts
- Categories support
- RSS feed
- Disqus commenting
- Google Analytics

---

## Local Development

```bash
# Clone the repository
git clone https://github.com/Choony1643/MyBlog.git

# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve

# Open http://localhost:4000/MyBlog/
